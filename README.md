# روائع التصميم للديكور — موقع تعريفي

موقع تسويقي عربي (RTL) لاستوديو ديكور داخلي فاخر في أبها.

## المعلومات
- **الاسم:** روائع التصميم للديكور
- **التصنيف:** تصميم وتنفيذ ديكور داخلي (Interior Designer)
- **المدينة:** أبها — حي شمسان، منطقة عسير
- **التقييم:** 4.9 على خرائط قوقل (١٧ مراجعة)
- **الجوال / واتساب:** 0501908463 · 966501908463

## سبب الاختيار
نشاط ديكور داخلي فاخر بتقييم مرتفع (4.9) وصور حقيقية ثرية للأعمال (مجالس، جدران مميّزة، صالة عرض) — مناسب لموقع تعريفي راقٍ يعرض الأعمال ويجمع طلبات عروض الأسعار.

## فكرة التصميم
اتجاه "كلاسيكي فاخر": أزرق كحلي عميق (#16213A) + ذهبي شامبين (#C7A35A) + كريمي دافئ (#F2ECE0). إحساس فلل ومجالس راقية يميّزه عن الأشقاء (المينيمال / الفحمي / الزمردي). خط العناوين **El Messiri** وخط النصوص **Tajawal**. معرض أعمال بارز، حركات دقيقة (ظهور تدريجي، تكبير الصور، هيدر يتقلّص).

## التقنيات
- HTML / CSS / JavaScript — **vanilla**، بدون أي مكتبات أو خطوة بناء.
- خطوط Google عبر `<link>` مع preconnect.
- صور حقيقية محلية مع `loading="lazy"`.
- IntersectionObserver للظهور التدريجي مع fallback، Lightbox خفيف بدون مكتبات.
- بيانات منظمة JSON-LD نوع `InteriorDesigner` مع `aggregateRating` 4.9/17.

## الصور
مصدر الصور: خرائط قوقل (صور النشاط نفسه — ملف المشروع). تم تنقيح المجموعة واستبعاد صورة لافتة نشاط آخر (HARDIUM) للإبقاء على صور النشاط الحقيقية فقط: مجالس/صالات داخلية، جدار رخام مميّز، صالة عرض، وواجهة المعرض.

## حالة الـ backend
**غير مربوط.** نموذج طلب عرض السعر تجريبي: يجهّز رسالة واتساب جاهزة عبر `wa.me/966501908463` ويحفظ نسخة في `localStorage` ويعرض إشعار نجاح (toast). لا توجد قاعدة بيانات أو خادم.
